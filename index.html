Algunos lenguajes de la Web

HTML, CSS, JavaScript

  • HTML

    HyperText Markup Language (lenguaje de marcas de hipertexto), hace referencia al lenguaje de marcado para la elaboración de páginas web.

  • CSS

    Cascading Style Sheets es un lenguaje usado para definir y crear la presentación de un documento estructurado escrito en HTML

  • JavaScript

    Es un lenguaje de programación interpretado, dialecto del estándar ECMAScript. Se define como orientado a objetos, basado en prototipos, imperativo, débilmente tipado y dinámico.
